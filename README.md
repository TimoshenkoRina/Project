Программа описывает простой вариант игр, основанных на рандоме, две из которых карточные. 
1. Сундучки - пользователь играет с двумя компьютерами, которые играют на рандом
2. Слоты, рулетка, блек джек - из реального казино в обычном представлении с коэффициентами и джекпотами
Так же есть система с кошельком и обновляющимся балансом
